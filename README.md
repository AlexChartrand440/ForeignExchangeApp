# ForeignExchangeApp
Sample app to try out react-native for iOS that includes:

* Navigation drawer
* Forms
* Push views
* Remote server request/response

## Usage
```
brew install node
npm install
open ForeignExchangeApp.xcodeproj
```

`CMD+R` to run the app from Xcode and then `CMD+R` to reload in the iOS Simulator.
